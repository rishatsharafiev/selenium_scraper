# EDADIL PARSER

### Install Java
```
apt-get update && apt-get upgrade
apt-get install default-jdk
java -version
```

### Download Selenium Standalone Server
https://www.seleniumhq.org/download/

### Run Selenium Standalone Server
```
java -jar selenium-server-standalone-*.jar
```
